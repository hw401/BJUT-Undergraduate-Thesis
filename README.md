## 北京工业大学本科生毕业论文模板
这是为北京工业大学的本科生毕业论文准备的Latex模板

环境：TexLive, VScode+Latex Workshop插件

使用方式：
将仓库copy到本地后用VScode打开即可，编译main.tex即可在根目录看到pdf版本的论文输出
